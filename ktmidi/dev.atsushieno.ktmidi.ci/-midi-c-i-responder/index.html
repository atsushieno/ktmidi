MidiCIResponder

class MidiCIResponder(val device: MidiCIDeviceInfo, sendOutput: (data: List<Byte>) -> Unit, muid: Int = Random.nextInt() and 0x7F7F7F7F)

This class is responsible only for one input/output connection pair

The argument sendOutput takes a sysex bytestream which is NOT specific to MIDI 1.0 bytestream (i.e. it should support sysex7 UMPs) and thus does NOT contain F0 and F7. Same goes for processInput() function.
